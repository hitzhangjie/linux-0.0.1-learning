从今天开始，开始阅读linux 0.0.1内核源代码，学习操作系统设计原理，加深自己对OS
知识的理解，也希望借此机会，帮助自己早些进入linux内核开发的大门。

源代码目录结构：

linux
	├── boot		核心引导代码，使用汇编编写，包括boot.s\head.s
	├── fs			文件系统代码
	├── include		头文件
	├── init		init进程，linux系统中的第一个进程，是所有进程的父进程
	├── kernel		系统调用
	├── lib			库代码
	├── Makefile	makefile文件，用于构建内核二进制可执行文件
	├── mm			内存管理
	└── tools		内核引导文件的制作工具
